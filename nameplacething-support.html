Need help with Name Place Thing? Find answers below or get in touch.

How to Play

Name Place Thing is the classic word game. Pick a letter, then write a name, place, animal, and thing that start with it. Same answers = 5 pts. Unique answers = 10 pts. Play 5 rounds and crown the winner!

Frequently Asked Questions

How do I create a game? Tap "Create game" on the home screen, enter your name, and tap "Create & wait." Share the 4-letter room code with friends.
How do I join a game? Tap "Join with code," enter your name and the 4-letter room code from the host. The code is case-sensitive (use uppercase).
My friend can't join my room. What should I check? Make sure you're both using the same 4-letter code (uppercase). Check your internet connection. If it still fails, the host can create a new room.
What do the difficulty levels mean? Easy = 1 letter (e.g., "A"). Medium = 2 letters (e.g., "AB"). Hard = 3 letters. The word must start with the first letter; the rest can appear anywhere.
How does scoring work? Same answer as someone else = 5 pts. Unique answer = 10 pts. Wrong or invalid answer = 0 pts. The first player to finish starts a 10-second timer for others.
How many players can play? Name Place Thing supports 2–8 players per game.
Does the app work offline? No. Online multiplayer requires an internet connection.
